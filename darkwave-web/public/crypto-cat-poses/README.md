# Crypto Cat Pose Images

Place transparent PNG files here with these exact filenames:

- `neutral.png` - Default grumpy face (head only)
- `angry.png` - Hands on hips, angry stance
- `pointing.png` - Pointing/scolding gesture
- `sideeye.png` - Side-eye dismissive look
- `walking.png` - Walking away
- `fist.png` - Power fist (HODL energy)
- `thumbsup.png` - Thumbs up approval
- `crossed.png` - Arms crossed defensive
- `facepalm.png` - Disappointed/regret

All images should be:
- Transparent background (PNG with alpha channel)
- Approximately 400x400px minimum
- Optimized for web (WebP with PNG fallback recommended)
